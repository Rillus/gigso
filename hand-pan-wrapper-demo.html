üîß JavaScript API

enableAudio()

Enables the audio context and starts Tone.js

disableAudio()

Disables the audio context

setKey(key, scale)

Changes the HandPan key and scale (e.g., setKey('F', 'major'))

setSize(size)

Changes the HandPan size ('small', 'medium', 'large')

getHandPan()

Returns the underlying HandPan component for advanced control

applyAudioEffects()

Applies current audio effect settings to the HandPan synthesizer

resetEffects()

Resets all audio effects to their default values
