Usage Instructions

Basic Usage

  • Plus Button (+): Increment BPM by step size
  • Minus Button (-): Decrement BPM by step size
  • Click Input: Enter edit mode for direct input
  • Enter Key: Commit changes in edit mode
  • Escape Key: Cancel changes in edit mode
  • Arrow Keys: Increment/decrement when input is focused

Keyboard Navigation

  • Tab: Navigate between controls
  • Shift+Tab: Navigate backwards
  • Space/Enter: Activate focused button
  • Arrow Up/Down: Change BPM when input is focused

Features

  • Plus/minus buttons for easy BPM adjustment
  • Direct text input with validation
  • Configurable min/max BPM and step size
  • Full keyboard navigation support
  • Accessibility features (ARIA labels, screen reader support)
  • Tone.js Transport integration
  • Global state management
  • Event-driven architecture
  • Input validation and error handling
  • Responsive design for mobile devices
