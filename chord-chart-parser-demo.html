Chord Chart Parser

How to use:

  • Paste your chord chart in the traditional format (chords above lyrics) into the input area
  • Fill in the song metadata (title, artist, key, etc.)
  • Click "Parse to JSON Format" to convert
  • The parsed lyrics will show chords integrated with lyrics using {chord} syntax
  • The full JSON object will be generated for use in the ukulele song library
  • Use "Copy JSON" to copy the result to your clipboard
